# Compiladores - Grupo 11

Bem-vindo à documentação do projeto de **Compiladores**.

Este projeto tem como objetivo desenvolver um compilador completo, abordando todas as etapas fundamentais do processo de compilação, desde a análise do código fonte até a geração de código final.

---

## Sobre o Projeto

O compilador é uma ferramenta responsável por traduzir código escrito em uma linguagem de alto nível para uma forma executável ou intermediária.

Neste projeto, implementamos:

-  Análise Léxica
-  Análise Sintática
-  Análise Semântica
-  Geração de Código

---

## Estrutura da Documentação

A documentação está organizada nas seguintes seções:

- **Sobre** → visão geral, escopo e objetivos do projeto  
- **Arquitetura** → detalhes das etapas do compilador  
- **Planejamento** → backlog e organização das sprints  
- **Evolução** → progresso das entregas (P1, P2)  
- **Testes** → estratégias, casos de teste e problemas encontrados  
- **Documentação** → como o projeto está estruturado e como executar  
- **Demo** → demonstração do funcionamento
